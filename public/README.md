# /public assets

このフォルダに静的アセットを配置します。

## 後から差し替え予定のもの

| パス | 用途 | サイズ目安 |
| --- | --- | --- |
| `/ogp-default.png` | サイト共通の OGP 画像 | 1200x630 |
| `/favicon.ico` | サイトファビコン | 32x32 / 48x48 |
| `/experiments/harassment-type/thumb.png` | ハラスメントタイプ診断のサムネイル | 1200x675 (16:9) |
| `/character/nunomaru.png` | ぬのまるキャラクター画像 (任意) | 任意 |

未配置でもサイトは動作します。`ExperimentCard` のサムネイルが見つからない場合は、
方眼紙＋歯車のプレースホルダーが自動表示されます。

画像が用意できたら、上記パスにそのまま置いてください。コードの変更は不要です。
