# /public/character/

ぬのまるキャラクター画像の置き場所。ここに PNG を置けば、後からコード側でそのまま参照できます。

## 推奨ファイル名 (置いた分だけ自動で使える想定)

| ファイル名 | 用途 | 推奨サイズ | 備考 |
| --- | --- | --- | --- |
| `nunomaru.png` | 基本立ち絵 (汎用) | 任意 | 透過 PNG 推奨 |
| `nunomaru-hero.png` | トップヒーロー横の大きめ表示 | 短辺 600〜800px / 縦長〜正方形 | 余白少なめ・透過 |
| `nunomaru-thinking.png` | 診断ページの案内役 | 短辺 400〜600px | 透過・腰から上など |
| `nunomaru-celebrate.png` | 結果ページ等 | 短辺 400〜600px | 透過 |
| `nunomaru-404.png` | 404 ページ | 短辺 300〜500px | 透過・小さめ |
| `nunomaru-favicon.png` | ファビコン用ベース画像 | 512x512 | 後で `/favicon.ico` に変換 |

## 画像仕様メモ

- フォーマット: **PNG (透過背景)** 推奨。背景がパーチメント色なので、白背景PNGを置くと白い四角が浮いてしまう。
- カラー: サイトベースが生成り＋真鍮系なので、線画は焦げ茶 (`#42301f` 付近) が馴染む。
- 解像度: 表示サイズの 1.5〜2 倍を目安に。Retina 対応用に少し大きめに。
- 透過の縁: アンチエイリアスが効いた状態で書き出すと、生成り背景と自然に馴染む。

## 配置場所の候補 (実装時にこちらでお伝えください)

- トップページのヒーロー右側 → `nunomaru-hero.png`
- 診断トップ・設問ページの案内役 → `nunomaru-thinking.png`
- 仮の結果ページ → `nunomaru-celebrate.png`
- 404 ページ → `nunomaru-404.png`
- フッターのコーナーにちょこっと → `nunomaru.png` (小さめ表示)

画像が用意でき次第、上記のうち「ここに置きたい」と教えてもらえれば、対応するページに差し込みます。
